# 鸣潮成就管理器

一个基于PySide6开发的鸣潮游戏成就管理工具，提供成就数据的获取和管理功能。

## 功能特性

### 🎯 成就管理
- **成就数据获取**：可从在线wiki获取最新的成就数据（非游戏内数据）
- **成就进度追踪**：记录和管理个人成就完成状态
- **成就分类管理**：按类别和版本等选项查看成就
- **成就搜索过滤**：快速查找特定成就

### 🎨 界面设计
- **现代化UI**：使用PySide6设计GUI界面
- **主题切换**：支持明暗两种主题模式
- **自定义头像**：支持切换游戏内不同角色头像和立绘

### 🔧 高级功能
- **数据导入导出**：支持JSON格式的数据备份和恢复（建议每个版本爬取前先导出记录备份）
- **多用户支持**：为不同用户保存独立的成就进度

## 安装说明

### 环境要求
- Python 3.8 或更高版本
- Windows 操作系统（推荐）

### 依赖安装
```bash
pip install -r requirements.txt
```

### 主要依赖
- PySide6 >= 6.4.0 - GUI框架
- requests >= 2.28.0 - HTTP请求库
- beautifulsoup4 >= 4.11.0 - HTML解析库

## 使用指南

### 启动应用
```bash
python main.py
```

### 基本操作
1. **成就数据管理**
   - 使用"成就管理"标签页查看和管理成就
   - 可以标记成就的完成状态（状态列：短按切换已完成或未完成，长按切换为暂不可获取）

2. **数据获取**
   - 使用"数据获取"标签页获取最新成就数据

### 数据格式
成就数据采用JSON格式存储，包含以下字段：
```json
{
  "绝对编号": "仅用于排序，每次覆盖数据会变化",
  "编号": "用于连接用户记录和基础数据，保持不变",
  "版本": "1.0",
  "名称": "成就名称",
  "描述": "成就描述",
  "奖励": "奖励内容",
  "是否隐藏": "隐藏",
  "获取状态": "未完成"
}
```
